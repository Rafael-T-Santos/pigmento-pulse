# Pigmento Pulse - Sistema de Consulta de Tintas

Sistema web profissional para cálculo de fórmulas, preços e gestão de tintas.

## Funcionalidades

- Consulta de fórmulas de cores (Iquine).
- Cálculo automático de preços via API.
- Gestão de bases e tamanhos.
- Cadastro de produtos no sistema ERP (Oracle).
- Interface responsiva e moderna.

## Tecnologias

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Shadcn UI.
- **Backend:** Python (Flask).
